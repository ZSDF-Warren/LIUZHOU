# 项目长期记忆

## 项目类型
柳州旅行景点地图 - 使用 Leaflet + 高德瓦片的 HTML 交互式地图，包含景点、螺蛳粉店、美食街等 55 个标记点。

## 小红书链接处理规则 (2026-03-24 添加)
当用户发送小红书链接时：
1. 先使用小红书技能读取帖子正文内容
2. 如果正文没有什么实质内容（大概率内容以文字方式出现在图片上），需要对图片进行文字识别（OCR）
3. 获取相关的完整内容后再进行后续处理

## 用户偏好 (2026-03-24 添加)
- **完成任务后默认提交到 git**：每次完成实质性任务改动后，自动 `git add` + `git commit` + `git push`，无需用户额外确认。

## 更新日志规则 (2026-03-24 添加)
- 地图页面右侧有一个"更新日志"按钮
- 每次有相关的更新提交到 git 后，需要在更新日志中说明更新的内容
- 更新日志应记录每次功能改进、新增景点、UI优化等变更

## 项目架构 (2026-03-24 重构)
模块化多文件结构：
- `index.html` — 精简 HTML 骨架
- `data/attractions.js` — 55个景点数据
- `data/itinerary.js` — 三天行程规划数据 (4/4~4/6)
- `data/config.js` — 分类/按钮配置
- `styles/main.css` — 所有样式
- `js/map-core.js` — 地图初始化、Marker管理、路线绘制
- `js/panels.js` — 天气/穿衣/行李/攻略面板
- `js/route.js` — 行程路线面板（左侧滑出时间轴）
- `js/app.js` — 应用入口、事件绑定
- `liuzhou_attractions_map.html` — 旧版单体文件（保留备份）

## 地图技术细节
- 高德瓦片 GCJ-02 坐标系，原始数据已是 GCJ-02 无需转换
- 景点数据存储在 `ATTRACTIONS` 数组 (data/attractions.js)
- 支持分类筛选：地标、自然、人文、夜景、民族、美食、螺蛳粉、美食街、🔥小红书热门
- 底部卡片横滑列表 + 详情抽屉 + 信息面板
- 左侧「行程路线」按钮 → 展开三天时间轴面板 + 地图虚线路线连线
- 行程规划：4/4 抵达 → 4/6 下午离开，以吃为主线穿插景点
